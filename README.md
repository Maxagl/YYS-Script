# 阴阳师脚本
## 声明
有被封的可能性，但是我自己用了两年了还没有被封

## 特色
1. 支持后台
2. 无鼠标冲突
3. 魂土双开
4. 狗粮只刷经验怪或金币怪
5. 小范围分辨率变化无影响
